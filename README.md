# react-music
使用react、ES6、webpack构建一个音乐播放器，包含进度条组件，播放器组件，header组件，列表组件等！


本代码库采用MIT协议，内容主要供学习和实践理解

# 运行方法
### 构建
```shell
npm build
```

### 启动
```shell
npm start
```

访问浏览器输入 http://localhost:3000 访问

### 备注
本学习实践采用的主要技术版本：
react               15.0.2
react-router        2.0.0"

webpack             1.13.0
